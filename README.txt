Prova de conceito para a alteração no artigo de game design 
Procura-se provar que é possivel construir um jogo do zero sem utilizar softwares comerciais
A linguagem python se mostrou muito eficiente se trabalhada junto com o módulo PyGame

Para compilar os programas faça o seguinte:

#Entre na pasta do projeto
  cd pygame-prova-conceitual/

#Compile usando o python
#Asteroids
  python asteroids/asteroids.py
#Plataform Scroller
  python plataform/plataform_scroller.py
#Shooter
  python shooter/shooter.py


O jogo asteroides foi utilizado para utilizar os conceitos de movimentação em todas as dimensões.
O jogo shooter foi utilizado para explorar o conceito de atirar juntamente com a movimentação
E o jogo Plataform Scroller(http://programarcadegames.com/) foi utilizado porque contem vários conceitos muito semelhantes ao projeto original utilizando Unity(https://github.com/ZehLuckmann/to_b)
